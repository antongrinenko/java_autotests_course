Пакет - это адрес для классов (как домашний адрес: город, улица, дом)
Класс - это как описание, чертёж чего-то (например чертёж или план дома)
Объект - это реальный объект, построенный по плану (реальный дом, автомобиль).
    Обънктов, сконструированных по одному классу может быть много (несколько
    домов, построенных по одному плану)
Родитель/Потомок - объект, может быть построен по нескольким чертежам (классам),
    которые по очереди дополняют друг друга. Базовый класс в данном случае называется
    Родитель, а следующий за ним класс - Потомок. Например, есть класс Животное со своим
    описанием (возраст, вес) и класс Кот со своим описанием (имя хозяина, порода). Кот
    построен по двум чертежам. Первоначально он Животное, а потом уже Кот. Здесь
    Животное - родительский класс, Кот - потомок от Животного

Типы данных:
    int - целые числа
    float - дробные числа
    String - строка
    boolean - правда или ложь

Область видимости переменных:
    private - доступны только внутри своего класса
    protected - доступны внутри класса и всем своим потомкам
    public - доступны абсолютно всем, кто интересуется

Функция
    Структура:
        [Область видимости] [Возвращаемый тип данных] [Имя]([Входные данные]) {
            [тело функции, какая-то логика]
            return [возвращаемое значение]
        }

    Пример:
        public int sumTwoNumbers(int a, int b) {
            return a + b;
        }
